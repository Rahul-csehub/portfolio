🌦️ Weather Monitor & Disaster Predictor
A Python-based system that monitors real-time weather parameters and uses data analysis to predict potential natural disasters like floods, droughts, and storms.

Fetches live weather data, processes it through threshold-based and pattern-detection logic, and alerts users with a risk-level summary — helping communities stay prepared before disaster strikes.

⚡ Example output:
📍 Region: Punjab | Rainfall: 180mm | Wind: 75km/h → ⚠️ HIGH FLOOD RISK — Evacuate low-lying areas
Python Weather API Data Analysis DBMS / SQL Disaster Prediction
